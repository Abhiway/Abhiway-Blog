What is Software ?

Software is a set of instructions, data or programs used to operate computers and execute specific tasks. In simple word software tells the computer to Perform the opearation.

Software Is Mainly Divided Into 2 Types They are System Software , Application Software

What is system software ?

System Software Is A Software Which Is Used To Create The Interface Between The User And Computer. Some Examples Can Be Operating Systems, Antivirus Softwares, Disk Formatting Softwares, Computer Language Translators Etc.

what is Application Software

Application Software Is Software Which Is Designed To Perform The Specific Operations And Tasks.
